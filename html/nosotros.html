Desarrollo de la página web

En este proyecto me encargué principalmente del desarrollo de la página web, trabajando en la estructura del sitio utilizando HTML, el diseño visual con CSS y algunas funciones interactivas con JavaScript y jQuery. También participé en la organización del contenido, la integración del carrusel de imágenes y el formulario de encuesta para recopilar opiniones de los usuarios. Mi objetivo fue crear una página clara, funcional y fácil de navegar que representara el ambiente del café.
