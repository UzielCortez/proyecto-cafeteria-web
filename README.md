# proyecto-cafeteria-web
Proyecto colaborativo desarrollado en equipo para la materia de Programación para internet. La página simula el sitio oficial de una cafetería e incluye menú interactivo, slider con bxSlider, mapa, formulario con validaciones en JavaScript y diseño responsive básico.
